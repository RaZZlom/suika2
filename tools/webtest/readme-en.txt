Suika2 Web Test Run Tool
========================

This program tests the Web version of Suika2.

# Preparation

* Put the following files in this folder:
  * data01.arc
  * index.html
  * index.js
  * index.wasm

# Run

* Run suika-webtest.exe
* A black window and a browser window will be shown
* Play game in the browser
* Close the black window when you finished a test
