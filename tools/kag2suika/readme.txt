ティラノビルダー作品をSuika2形式に変換するプログラム
====================================================

ティラノビルダー作品のフォルダをkag2suika.exeにドロップすると、suikaフォルダが生成されます。

他エンジンからSuika2への移植は推奨されておりませんので、あくまで参考のためにご利用ください。

下記の基本的な機能だけ変換してくれます:
 - メッセージ
 - セリフ
 - 背景画像
 - キャラ画像
 - BGM
 - SE
 - ジャンプ

手動で次の作業が必要です:
 - Suika2のコンフィグフォルダconfはSuika2のサンプルゲームからコピーしてください
 - 画像サイズは一括変換をかけてください
 - システム画像は手動で変更してください
