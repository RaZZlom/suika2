What Suika2 Is

Suika2 is a visual novel development suite with an open-source philosophy, a true cross-platform design, and production-ready quality.

Where Suika2 Aims To Be

Suika2 will be the standard of visual novel development for the 2020s and later.

What Suika2 Values

  • Simple: Anyone can easily create visual novels.
  • Fast: We build apps with fast native technologies only.
  • Free: We are an open source project and we make our technology public as pro-bono while we respect other developers' intellectual properties.
  • Diverse: Games can be developed on a variety of platforms and run on any device.
  • Cultural: We believe that creation of information is the true value of humanity.
  • Earning: Game developers can publish their games on stores and get income.
  • Prosperity: We aim to create a world where people of any birth can make a living with a single computer.

System Requirements

  • Microsoft Windows
    • Windows 10 (32bit/64bit)
    • Windows 11
    • We support Arm64 including Surface Pro X
  • Mac
    • macOS 10.13 High Sierra
    • macOS 10.14 Mojave
    • macOS 10.15 Catalina
    • macOS 11 Big Sur
    • macOS 12 Monterey
    • macOS 13 Ventura
    • macOS 14 Sonoma
