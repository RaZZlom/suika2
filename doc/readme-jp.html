このソフトについて

  • ビジュアルノベルの開発ツールです
  • 開発はWindows/macOS/Linuxで行えます
  • 作成したゲームの動作環境はWindows/macOS/Webブラウザ/iOS/Android/Linuxです
  • 商用実績があるのはWindows/iOS/Androidです
  • スマートフォンはPCより性能が低いですが、それでも確実に移植して動作可能とするため、演出を控え気味にしてあります
  • 近いうちに家庭用ゲーム機へ公式移植の予定です
  • ユーザ参加型で開発しておりますので、ぜひバグ報告や機能要望をお願いいたします

  • ZIPファイルを展開したフォルダ suika2 をそのままご使用ください
  • 不要になりましたらフォルダごと削除してください

サンプルゲームの起動方法

  • Windowsでは、
    • suika-pro.exe を開いてください
    • セキュリティ警告が表示された場合、「詳細情報」をクリックしてから「実行」をクリックしてください
  • Macでは、
    • 展開されたフォルダ suika2 を開いてください (※1)
    • tools フォルダを開いてください
    • mac-pro.dmg を開いて、中に入っているアプリ Suika2 Pro for Creators を、ゲームフォルダである suika2 (※1)にコピーします
    • ゲームフォルダ suika2 の中で、アプリ Suika2 Pro for Creators を開いてください

ライセンス

  • アプリ本体と開発ツールはMITライセンスです
  • 利用ライブラリのライセンス表示は実行ファイルに埋め込んであるため不要です (zlib, libpng, jpeg, bzip2, libwebp, libogg, libvorbis, freetype)
  • サンプルゲームのフォントは再配布可能ですが別ライセンスですので、フォントフォルダにあるライセンスファイルの内容を、作品のREADMEやエンドロールで表示してください
  • サンプルゲームの音源のうちBGMの「蝉の声」と「海岸の音」は二次利用できません
