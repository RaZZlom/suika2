    Linux/Chromebook

    ソースコードからビルド (推奨):

    sudo apt-get install -y git build-essential libasound2-dev libx11-dev mesa-common-dev qt6-base-dev qt6-multimedia-dev libwebp-dev
    git clone https://github.com/suika2engine/suika2.git
    cd suika2
    make
    sudo make install
    suika2

    Ubuntuパッケージのインストール (22.04 LTSのみ):

    sudo add-apt-repository ppa:ktabata/ppa
    sudo apt update
    sudo apt install suika2
    suika2

    動作環境

    Windows

    • 制作ツール Suika2 Pro Desktop (Windows)
      • OS: Windows 11, Windows 10 (Windows 10は32bit/64bit対応)
      • CPU: Intel/AMD/Arm64
      • GPU: Direct3D 9 (今後DirectX 12に移行予定)
    • 出力ゲーム (Windows)
      • OS: Windows 11, Windows 10, Windows 8.1, Windows 8, Windows 7, Windows Vista, Windows XP (Windows 11以外は32bit/64bit対応)
      • CPU: Intel/AMD/Arm64
      • GPU: Direct3D 9 (今後DirectX 12を選択可能にする予定)
